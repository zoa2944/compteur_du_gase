Achats

Cette page sert à :
  • enregistrer l'achat d'un membre ;
  • corriger ou annuler un achat déjà effectué ;
  • obtenir les informations sur les derniers achats effectués (pour cela commencer un nouvel achat) ;
  • obtenir les informations sur le solde restant d'un foyer (pour cela commencer un nouvel achat).
Pour annuler un achat erronné, une façon de faire est de refaire le même achat en mettant une quantité négative à la place. Cela réapprovisionne le compte et le stock correctement.
